## ⚙️ Getting Started

### Prerequisites

* Node.js 18+
* npm 9+

### Installation

```bash
git clone https://github.com/Omjagdal/3d-portfolio.git
cd 3d-portfolio
npm install
npm run dev
```

Open: `http://localhost:5173`

## 🎨 Customization Guide

To personalize this portfolio:

* Update content → `src/components/`
* Edit project data → `src/data/`
* Modify styles → `src/components/styles/`
* Customize 3D scene → `src/components/Character/`
* Update animations → GSAP utilities
